================================================================================
WHISKER & PAWS - AI-POWERED AD VARIATION GENERATOR
Documentation & Process
================================================================================

PROJECT OVERVIEW
----------------
An AI-powered web application that takes a single ad concept for "Whisker & Paws"
(a DTC cat subscription box) and generates 5 platform-optimized ad variations
with AI-generated images.

TECH STACK
----------
- Framework: Next.js 15 (App Router)
- Language: TypeScript
- Styling: Tailwind CSS
- AI Text: Google Gemini 2.0 Flash
- AI Images: Google Imagen 3
- Hosting: Vercel

================================================================================
TOOLS & PROMPTS USED
================================================================================

1. TEXT GENERATION (Gemini 2.0 Flash)
-------------------------------------
Model: gemini-2.0-flash
Configuration: JSON response mode for structured output

PROMPT:
"""
You are an expert advertising copywriter specializing in direct-to-consumer brands.
Generate ad copy variations for a cat subscription box called "Whisker & Paws".

INPUT CONCEPT: {concept}

Generate ad copy for these 5 platforms with these EXACT specifications:

1. GOOGLE SEARCH ADS (text only):
   - 3 headlines (max 30 characters EACH)
   - 2 descriptions (max 90 characters EACH)
   - Display URL path (max 15 characters)
   - NO emojis allowed

2. META (Facebook/Instagram):
   - Primary text (max 125 characters, hook-driven)
   - Headline (max 27 characters)
   - Emotional language, 1-2 emojis allowed

3. TIKTOK:
   - Ad text (max 100 characters)
   - Casual, authentic tone
   - Emojis and trending language allowed

4. DISPLAY/BANNER:
   - Headline (max 30 characters)
   - Description (max 90 characters)
   - CTA (max 15 characters)

5. YOUTUBE:
   - Headline (max 40 characters)
   - Description (max 90 characters)

Return as valid JSON structure.
"""

2. IMAGE GENERATION (Imagen 3)
------------------------------
Model: imagen-3.0-generate-002
Aspect ratios configured per platform

META IMAGE PROMPT (1080x1080, 1:1):
"""
Create a professional advertising image for a cat subscription box brand called
"Whisker & Paws". Show a happy, photogenic cat interacting with colorful toys.
Include an attractive subscription box in the scene. Bright, warm lighting.
Clean background. High-end product photography style. NO TEXT IN IMAGE.
"""

TIKTOK IMAGE PROMPT (1080x1920, 9:16):
"""
Create a vibrant, attention-grabbing vertical image for a TikTok ad. Dynamic,
energetic composition. Cute cat in excited or playful pose. Colorful toys and
treats. Bright, saturated colors that pop on mobile. Fun, youthful aesthetic.
NO TEXT IN IMAGE.
"""

DISPLAY IMAGE PROMPT (300x250, 1:1):
"""
Create a clean, impactful banner ad image. Simple, focused composition. Cute cat
face or cat with single toy. Clean background for text overlay. High contrast
for small size visibility. NO TEXT IN IMAGE.
"""

YOUTUBE IMAGE PROMPT (1280x720, 16:9):
"""
Create a cinematic, wide-format image. Happy cat in cozy home environment.
Subscription box with toys spilling out. Warm, inviting setting. Soft, natural
lighting. Premium lifestyle photography feel. NO TEXT IN IMAGE.
"""

APPROACH & FUTURE OPTIMIZATIONS (250 words)
================================================================================

APPROACH:
I built this MVP prioritizing speed-to-ship while maintaining quality. The core
architecture uses Next.js App Router for seamless Vercel deployment with API
routes handling the AI orchestration.

The generation flow is optimized for efficiency: text generation happens in a
single Gemini call using JSON mode for reliable structured output, while all 4
images generate in parallel via Imagen 3 with platform-specific aspect ratios.
This parallel approach reduces total generation time from ~80s sequential to
~25s concurrent.

Each platform has researched specifications baked in - character limits, image
dimensions, and tone guidelines that match real advertising requirements. The
prompts are engineered to produce copy that matches each platform's native feel:
professional for Google, emotional for Meta, casual for TikTok.

FUTURE OPTIMIZATIONS:
1. A/B Testing Integration: Generate multiple copy variants per platform with
   headline/description permutations for systematic testing
2. Brand Asset Upload: Allow custom logos, color palettes, and product images to
   be incorporated into generated visuals
3. Performance Scoring: Use historical ad performance data to score/rank
   generated variations by predicted CTR
4. Video Generation: Extend to short-form video for TikTok/Reels using
   AI video generation
5. Audience Targeting Suggestions: Generate persona-specific variations with
   recommended targeting parameters
6. Export to Ad Platforms: Direct API integration with Google Ads, Meta Business
   Suite for one-click campaign creation
7. Iteration Memory: Store successful ad patterns to inform future generations

================================================================================
SETUP INSTRUCTIONS
================================================================================

1. Clone the repository
2. Install dependencies: npm install
3. Copy .env.example to .env.local
4. Add your GEMINI_API_KEY to .env.local
5. Run development server: npm run dev
6. Open http://localhost:3000

For production deployment:
1. Push to GitHub
2. Import project to Vercel
3. Add GEMINI_API_KEY to Vercel Environment Variables
4. Deploy
